# 🌍 TerraMind  
### AI-Powered Climate Impact Analyzer (Terminal-Based)

TerraMind is a terminal-based artificial intelligence project designed to analyze how daily human habits affect the environment. The system predicts environmental impact levels and provides personalized climate solutions to promote sustainable decision-making.

This project connects artificial intelligence with environmental science to address global issues such as climate change and global warming.

---

## 🚀 Features

- 🌱 Terminal-based user interface  
- 🤖 Machine learning classification model  
- 📊 Predicts impact level: Low / Medium / High  
- 💡 Provides personalized sustainability suggestions  
- 🧠 Built using Python and Scikit-learn  
- 🌍 Focused on climate awareness and carbon footprint reduction  

---

## 🧠 How It Works

1. The user inputs:
   - Transport type (car, bus, bike, etc.)
   - Electricity usage (low, medium, high)
   - Recycling habits (yes/no)

2. The trained ML model analyzes the data.

3. The system predicts environmental impact level.

4. TerraMind generates practical solutions to reduce environmental damage.

## 🛠 Installation

1. Clone the repository:

git clone https://github.com/yourusername/terramind.git  
cd terramind  

2. Install dependencies:

pip install -r requirements.txt  

3. Train the model:

python model/train_model.py  

4. Run the program:

python main.py
